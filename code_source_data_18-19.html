Timing of copy neutral loss-of-heterozygosity events in endometrial glands

Markdown file to present the methods to examine timing of copy neutral loss-of-heterozygosity events in endometrial glands. Before you start the following analyses, please convert the corresponding excel source data files to tab-delimited text files. The source data files are provided in our paper (Yamaguchi et al. Nat Commun in press).

Estimation of binomial probabilities that public mutations occurred before or after copy neutral loss-of-heterozygosity event based on single gland

mafData$binom_pre <- mapply(
  function(a, b, c) {
    return(dbinom(a, b, prob=c))},
  mafData$num_alt_read, mafData$depth, mafData$purity)

mafData$binom_post <- mapply(
  function(a, b, c) {
    return(dbinom(a, b, prob=c))},
  mafData$num_alt_read, mafData$depth, mafData$purity/2)

write.table(mafData, "source_data_18_binom_prob.txt", quote=F, col.names=T, row.names=F, append=F, sep="\t")

Estimation of timing of copy neutral loss-of-heterozygosity events and their confidence intervals in endometrial glands of subject 36

sample_age <- 50 + 5/12
prop_public <- 0.9760442
var_prop_public <- 8.412926e-06

simData$time_chr3 <- sample_age * prop_public * simData$prop_pre_chr3
simData$var_chr3 <- var_prop_public * simData$var_prop_pre_chr3 + prop_public^2 * simData$var_prop_pre_chr3 + simData$prop_pre_chr3^2 * var_prop_public
simData$time_chr3_lower95 <- sample_age * (prop_public * simData$prop_pre_chr3 - 1.96 * sqrt(simData$var_chr3))
simData$time_chr3_upper95 <- sample_age * (prop_public * simData$prop_pre_chr3 + 1.96 * sqrt(simData$var_chr3))

simData$time_chr10 <- sample_age * prop_public * simData$prop_pre_chr10
simData$var_chr10 <- var_prop_public * simData$var_prop_pre_chr10 + prop_public^2 * simData$var_prop_pre_chr10 + simData$prop_pre_chr10^2 * var_prop_public
simData$time_chr10_lower95 <- sample_age * (prop_public * simData$prop_pre_chr10 - 1.96 * sqrt(simData$var_chr10))
simData$time_chr10_upper95 <- sample_age * (prop_public * simData$prop_pre_chr10 + 1.96 * sqrt(simData$var_chr10))

res <- data.frame(
  Chromosome=c("chr3", "chr10"),
  Age=c(mean(simData$time_chr3), mean(simData$time_chr10)),
  Lower_95=c(mean(simData$time_chr3_lower95), mean(simData$time_chr10_lower95)),
  Upper_95=c(mean(simData$time_chr3_upper95), mean(simData$time_chr10_upper95)))
