Associations between clinical variables and burden of driver mutations in endometrial glands

Markdown file to present the methods to examine associations between clinical variables and burden of driver mutations in endometrial glands. Before you start the following analyses, please convert the corresponding excel source data files to tab-delimited text files. The source data files are provided in our paper (Yamaguchi et al.Nat Commun in press).

Pearson correlation coefficients between clinical variables and burden of driver mutations

name <- c("Age", "Cumulative number of menstrual cycles")
clin_var <-c("age", "cum_num_menstrual_cycles")
r_coef <- vector(mode="numeric", length(clin_var))
p_value <- vector(mode="numeric", length(clin_var))

for (i in 1:length(clin_var)) {
  clin_data <- mutBurd[, clin_var[i]]
  res <- cor.test(clin_data, mutBurd$driver_mutation_burden)
  r_coef[i] <- res$estimate
  p_value[i] <- res$p.value
}

result_cor <- data.frame(
  clinical_variable=name,
  r=r_coef,
  p_value=p_value)
